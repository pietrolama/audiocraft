# Frontend AudioCraft

Frontend Next.js 15 per AudioCraft Text-to-Audio Generator.

## Setup Sviluppo

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000)

## Build Produzione

```bash
npm run build
npm start
```

## Tecnologie

- **Next.js 15**: Framework React con App Router
- **TypeScript**: Type safety
- **TailwindCSS**: Styling moderno
- **React Hook Form + Zod**: Form validation
- **IndexedDB**: Storage locale per cronologia
- **SSE**: Progress updates in tempo reale

## Struttura

- `src/app/`: Pages e routing
- `src/components/`: Componenti React
- `src/app/api.ts`: Client API
